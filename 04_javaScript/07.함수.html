함수

함수를 선언하는 방법

  1. 선언적 함수(Function Declaration): function키워드, 함수명
  2. 익명 함수(Function Expression): 변수에 익명함수를 대입하는 방식
  3. 화살표 함수(Arroww Function): 익명함수를 간단히 작성하는 방법(ES6버전부터 추가)
  4. 즉시 실행 함수 life(Immediately Invoked Function Expression)
    • 선언과 호출
    • 변수를 캡슐화처리 유용

* 선언적 함수

        Function 함수명([매개변수,...]) {
            해당 함수 호출시 실행할 코드;
            [return 결과값;]
        }

* 익명 함수

        변수|속성 = function([매개변수,..]) {
            해당 함수 호출시 실행할 코드;
            [return 결과값;]
        }

        - 주로 이벤트 핸들러 작성시 사용

* 화살표 함수(Arrow Function)

        - 본문에 리턴만 있을때
          변수 = ([매개변수,..]) => 리턴값;

        - 본문이 여러줄 일때
          변수 = ([매개변수,..]) => {
            해당 함수 호출시 실행할 코드;
            [return 결과값;]
          }

* 즉시 실행 함수

        - 정의하면서 즉시 실행되는 함수
        - 단 한번만 호출됨
          ([변수] = function([매개변수,..]) {
            해당 함수 호출시 실행할 코드;
          })(인자,...);

* 매개변수 지정 방법

        - 매개변수가 없을 경우:         () => 리턴값;       (소괄호 반드시 있어야함)
        - 매개변수가 1개일때:           x => 리턴값;        (소괄호 생략 가능)
        - 매개변수가 1개 이상일때:      (x, y) => 리턴값;   (소괄호 반드시 있어야됨)
