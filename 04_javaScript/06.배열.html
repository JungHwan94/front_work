배열

- 변수 선언시 별도의 자료형 지정하지 않음
- 어떤 자료형의 값들이어도 모두 하나의 배열에 담을수 있음
- 크기의 제약 없음
- 존재하지 않는 인덱스를 참조해도 오류가 나지 않음. undefined리턴


* 배열의 선언

* 배열관련 메소드

1) 배열. indexOf(찾고자하는 요소): 배열에서 해당 요소의 인덱스 번호 반환


2) 배열.concat(배열, 배열, ...): 배열 여러개를 결합하고자 할때


3) 배열.sort(): 배열에 담긴 값들을 오름차순 정렬을 하고자 할때

4-1) 배열.push(추가할 요소): 배열의 맨뒤에 요소를 추가하고 배열의 크기를 반환
4-2) 배열.pop(): 배열의 맨뒤의 요소를 제거하고 제거된 요소 반환


5-1) 배열.unshift(추가할 요소): 배열의 맨앞에 요소를 추가하고 배열의 크기를 반환
5-2) 배열.shift(): 배열의 맨앞의 요소를 제거하고 제거된 요소 반환


5-1) 배열.slice(시작인덱스, 끝인덱스): 배열의 시작인덱스부터 끝인덱스까지 추출해주는 메소드
5-2) 배열.splice(시작인덱스, 제거수[, 추가할 값]): 배열의 요소들을 추출하여 제거 및 추가해줌. 반환값 제거한 요소
